# 对话气泡聊天应用

这是一个简单的网页聊天应用，支持两个角色之间的对话，并以气泡形式展示消息。应用采用纯前端实现，具有简洁的界面和流畅的交互体验。

## 功能特点

- 支持角色A和角色B之间的对话切换
- 消息以气泡形式展示，不同角色有不同的样式
- 支持通过按钮或回车键发送消息
- 自动滚动到最新消息
- 响应式设计，适配不同屏幕尺寸

## 项目结构

```
├── index.html    # 主页面HTML文件
├── style.css     # 样式表文件
├── script.js     # JavaScript脚本文件
└── README.md     # 项目说明文档
```

### 文件说明

- `index.html`: 定义了应用的基本HTML结构，包含聊天容器、角色切换按钮、消息显示区域和输入区域
- `style.css`: 包含所有样式定义，实现了消息气泡、按钮动画等视觉效果
- `script.js`: 实现了应用的核心功能，包括角色切换、消息发送和显示等交互逻辑

## 使用方法

1. 直接在浏览器中打开`index.html`文件即可运行应用
2. 点击右上角的角色切换按钮可以在角色A和角色B之间切换
3. 在底部输入框中输入消息，点击发送按钮或按回车键发送消息
4. 消息会以对应角色的气泡样式显示在聊天区域

## 维护指南

### 样式修改

- 消息气泡样式：修改`style.css`中的`.message.role-a`和`.message.role-b`相关样式
- 按钮样式：修改`.role-button`和`#sendButton`相关样式
- 布局调整：修改`.chat-container`和`.messages`相关样式

### 功能扩展

- 添加新功能时，建议在`script.js`中按模块组织代码
- 所有DOM操作都在`DOMContentLoaded`事件监听器中进行
- 新增UI元素时，在`index.html`中添加相应结构，并在`style.css`中定义样式

## 注意事项

- 项目使用原生JavaScript实现，无需额外依赖
- 样式采用Flexbox布局，确保浏览器兼容性
- 消息内容支持长文本自动换行
